**CPU所需访问信息已在Cache中的概率称为Cache命中率**

$$命中率定义为：H = \frac{N_{c}}{N_{c} + N_{m}}$$
$N_{c}$ ：Cache命中次数
$N_{m}$：访问主存的次数（未命中次数）
**命中时**：CPU直接从Cache读取数据，耗时为命中时间$T_c$ (访问Cache的时间)
**未命中时**：CPU需先从主存读取包含目标数据的一个主存块送入Cache，再将所需的数据送至CPU，总耗时为：$T_m+T_c$ 其中$T_m$ 称为缺失损失，即从主存调入一个块所需的时间

Cache-主存系统平均访问时间$T_a$为：
$$T_a = HT_c + (1 - H)(T_m + T_c) = T_c + (1 - H)T_m$$
